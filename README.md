# SOLID-node

I developed an application for gym check-ins. Here, you will learn about SOLID principles, design patterns, using Docker to set up the database, JWT and refresh tokens, RBAC, and various other concepts.

# App

## Requisitos funcionais

- [ ] It should be possible sign-up;
- [ ] It should be possible authenticate;
- [ ] It should be possible to get user profile
- [ ] It should be possible to get check-ins for user logged
- [ ] It should be possible to get check-ins history
- [ ] It should be possible to get nearly gyms
- [ ] It should be possible check-in
